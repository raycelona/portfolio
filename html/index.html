Work

CVS Health

I'm currently employed by CVS Health, headquartered in Woonsocket RI. I'm on a team of developers working to build, upgrade and maintain the ExtraCare portion of cvs.com.

Working in an Agile/Scum environment, I utilize the latest versions of Angular (6-10), HTML5, CSS3 with SASS and JavaScript ES6. Using Gitlabs as source control and CI/CD pipelines through Jenkins I work to deploy new features and security upgrades on a weekly basis.

Optum

Optum is the technology hub for United Health Group and is headquartered in the Fenway area of Boston, MA. My time there was spent specializing on HTML and CSS layouts and utilizing JavaScript and AngularJS to create new customer facing healthcare and healthcare facilities software.

Fidelity Investments

Fidelity Investments in Smithfeild RI had a need to create a brand new, responsive mobile site for their advisor site and I was the first UI developer to be brought on for this effort.

Utilizing HTML5, CSS3 and JavaScript I constructed the responsive advisor site after much collaboration with business stakeholders, project managers and UX designers.
